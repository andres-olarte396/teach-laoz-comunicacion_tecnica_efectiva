# Rúbrica para Documentación Técnica

| Criterio                       | Excelente (3)                            | Bueno (2)                                 | Mejorable (1)        |
| ------------------------------ | ---------------------------------------- | ----------------------------------------- | -------------------- |
| Claridad y precisión           | Mensaje claro, sin ambigüedades          | Mayormente claro, algunos detalles faltan | Confuso o incompleto |
| Estructura y organización      | Secciones bien definidas y ordenadas     | Estructura aceptable                      | Desorganizado        |
| Ejemplos y casos de uso        | Ejemplos relevantes y completos          | Ejemplos presentes pero limitados         | Sin ejemplos         |
| Uso de formato y herramientas  | Markdown, diagramas, enlaces bien usados | Formato correcto pero mejorable           | Formato pobre        |
| Actualización y mantenimiento  | Documentación actualizada y versionada   | Actualización parcial                     | Desactualizada       |
| Adaptación al público objetivo | Lenguaje y nivel técnico adecuado        | Parcialmente adaptado                     | No adaptado          |
