## Ejemplo de documentación

README para el proyecto conversor_temperatura:

````markdown
# Conversor de Temperatura

Este proyecto permite convertir valores entre Celsius y Fahrenheit.

## Uso

```python
from conversor_temperatura import celsius_a_fahrenheit, fahrenheit_a_celsius
print(celsius_a_fahrenheit(0))  # 32.0
print(fahrenheit_a_celsius(32)) # 0.0
```
````
