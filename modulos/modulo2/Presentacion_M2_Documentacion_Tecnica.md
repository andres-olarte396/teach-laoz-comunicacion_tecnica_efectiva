# Presentación Módulo 2: Documentación técnica y diseño instruccional

## Objetivo

Transformar conocimiento técnico en documentación clara y experiencias de aprendizaje para desarrolladores.

## Temas clave

- Estructura de documentación técnica
- Uso de Markdown y buenas prácticas
- Diseño de tutoriales y laboratorios prácticos

## Ejercicios

- Redactar documentación de una función, API o proceso
- Diseñar un tutorial paso a paso

## Recursos

- Ejemplos de documentación en GitHub
- Plantillas de README

## Evaluación

- Revisión entre pares
- Presentación y retroalimentación
