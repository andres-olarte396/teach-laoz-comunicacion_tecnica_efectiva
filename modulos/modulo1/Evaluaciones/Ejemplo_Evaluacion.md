# Evaluación Módulo 1: Comunicación técnica en equipos

## Checklist de observación

- [ ] El feedback es claro y específico
- [ ] Se usan ejemplos concretos
- [ ] El tono es respetuoso y empático
- [ ] Se proponen soluciones o alternativas
- [ ] Se adapta el mensaje al nivel del receptor

## Rúbrica de feedback técnico

| Criterio | Excelente (3)                    | Bueno (2)            | Mejorable (1)           |
| -------- | -------------------------------- | -------------------- | ----------------------- |
| Claridad | Mensaje claro y sin ambigüedades | Mayormente claro     | Confuso o ambiguo       |
| Empatía  | Tono positivo y constructivo     | Tono neutro          | Tono crítico o negativo |
| Utilidad | Feedback útil y accionable       | Útil pero incompleto | Poco útil o irrelevante |
| Ejemplos | Usa ejemplos concretos           | Ejemplos limitados   | Sin ejemplos            |
