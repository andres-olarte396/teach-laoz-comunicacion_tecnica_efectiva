# Actividad Módulo 1: Simulación de revisión de código y feedback

## Descripción

Realiza una revisión de código sobre un fragmento de código propuesto por un compañero. Analiza claridad, buenas prácticas y comunica tus observaciones de forma constructiva.

## Pasos

1. Lee el fragmento de código y detecta posibles mejoras (nombres, estructura, errores, claridad).
2. Escribe comentarios en estilo GitHub, usando lenguaje claro y empático.
3. Expón tu feedback en una breve reunión simulada (5 min).
4. Recibe retroalimentación sobre tu comunicación.

## Ejemplo de código para revisión

```python
def calcular_total(productos):
 total = 0
 for p in productos:
  total += p['precio']
 return total
```

Checklist de revisión técnica:

- ¿Los nombres de variables son claros y descriptivos?
- ¿Se manejan posibles errores (por ejemplo, si falta la clave 'precio')?
- ¿El código es eficiente y fácil de mantener?
