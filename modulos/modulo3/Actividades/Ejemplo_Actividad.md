# Actividad: Creación de un diagrama de arquitectura

## Descripción

Crea un diagrama de arquitectura para el siguiente sistema de ejemplo, utilizando la herramienta de tu preferencia.

**Sistema:** Plataforma web para gestión de tareas colaborativas.

## Instrucciones

- Identifica los componentes principales (frontend, backend, base de datos, API, autenticación).
- Representa la interacción entre los componentes.
- Incluye leyenda y breve explicación del diagrama.
- Comparte tu diagrama con el grupo y recibe retroalimentación.
