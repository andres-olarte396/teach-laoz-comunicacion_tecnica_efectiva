# Presentación Módulo 4: Herramientas digitales colaborativas y visualización técnica

## Objetivo

Dominar herramientas para documentar, colaborar y presentar contenido técnico profesional.

## Temas clave

- Uso de GitHub y control de versiones
- Markdown y diagramas técnicos (Mermaid, PlantUML)
- Diseño visual de presentaciones técnicas

## Ejercicios

- Crear y documentar un repositorio colaborativo
- Rediseñar una slide técnica y crear un diagrama de flujo

## Recursos

- Guía de Markdown
- Ejemplo de workflow en GitHub

## Evaluación

- Revisión de repositorio
- Calidad visual y claridad técnica
