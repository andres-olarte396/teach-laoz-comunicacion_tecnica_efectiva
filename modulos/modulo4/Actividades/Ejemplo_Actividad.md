# Actividad: Creación y documentación de un repositorio colaborativo

## Descripción

Crea un repositorio en GitHub para un proyecto de ejemplo y documenta el flujo de trabajo colaborativo usando issues, pull requests y un tablero Kanban.

## Instrucciones

- Configura el repositorio y agrega un archivo README.md con descripción y objetivos.
- Crea al menos dos issues y simula su resolución mediante pull requests.
- Usa el tablero de proyectos de GitHub para visualizar el avance.
- Documenta el proceso con capturas de pantalla y comentarios.
