# Curso Comunicación Técnica Efectiva para Desarrolladores

Este repositorio contiene el programa, presentaciones y plantillas para el curso "Comunicación Técnica Efectiva para Desarrolladores". El objetivo es mejorar la comunicación, documentación y exposición técnica en equipos de desarrollo.

## Navegación y estructura

Para consultar la estructura completa del curso, módulos, actividades, materiales y evaluaciones, revisa el archivo [`INDICE.md`](./INDICE.md).

## Estructura del repositorio

- `Curso_Estructurado_Comunicacion_Tecnica.md`: Programa completo del curso, módulos y sesiones.
- `Presentacion_M1_Comunicacion_Tecnica.md` a `Presentacion_M6_Evaluacion_Mejora.md`: Esquemas de presentaciones por módulo.
- `Plantilla_README_Tecnico.md`: Plantilla base para documentación de proyectos técnicos.
- `Rubrica_Documentacion_Tecnica.md`: Rúbrica para evaluar documentación técnica.
- `Checklist_Revision_Codigo.md`: Lista de verificación para revisiones de código.
- `README_Plantillas.md`: Resumen de recursos y plantillas disponibles.

## ¿Cómo usar este repositorio?

1. Revisa el programa del curso y adapta los módulos según tu equipo o contexto.
2. Utiliza las presentaciones como base para crear diapositivas en PowerPoint o Google Slides.
3. Aplica la plantilla README y la rúbrica en tus proyectos técnicos para mejorar la documentación.
4. Usa la checklist en revisiones de código para asegurar calidad y buenas prácticas.
5. Personaliza los recursos según el stack tecnológico y necesidades de tu equipo.
